# ⚽ Football Live Action Counter

## 📌 Overview
This project is an **Excel-based tool** designed to track and count live football actions during a game in real time. It allows analysts and coaches to record key actions such as shots, crosses, passes, tackles and interceptions, providing an instant visual summary through an automatically updated chart.

## 🔍 Key Features
- 🎯 Live tracking of key football actions during a game
- 📊 Automatic chart update as actions are recorded
- ⚽ Tracks: **Shots, Crosses, Passes, Tackles, Interceptions**
- 🎨 Color-coded action buttons for quick and easy input
- 📈 Instant visual summary of actions per game

## 🚀 How to Use
1. Download the `1.7.2 João Almeida.xlsm` file
2. Open it using **Microsoft Excel**
3. Enable **macros** when prompted
4. During the game, click the **colored action buttons** to record each action
5. The chart updates **automatically** in real time
